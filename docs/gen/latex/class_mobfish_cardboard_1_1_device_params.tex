\doxysubsection{Detailed Description}

\begin{DoxyItemize}
\item Message describing properties of a VR head mount device (HMD) which uses an interchangeable smartphone as a display (e.\+g. Google Cardboard). 
\end{DoxyItemize}

While some properties are certain (e.\+g. inter\+\_\+lens\+\_\+distance), others represent nominal values which may be refined depending on context (e.\+g. viewport\+\_\+angles).

Lengths are in meters unless noted otherwise. Fields are {\itshape required} unless noted otherwise.

Some context on why this set of parameters are deemed necessary and sufficient\+:
\begin{DoxyItemize}
\item FOV scale can be reasonably approximated from lens-\/to-\/screen distance and display size (i.\+e. knowing lens focal length isn\textquotesingle{}t crucial).
\item Lenses are assumed to be horizontally centered with respect to display.
\item The display is not necessarily vertically centered. For interchangeable phones where the device rests against a tray, we can derive the vertical offset from tray-\/to-\/lens height along with phone-\/specific bezel and screen sizes (supplied separately). 
\end{DoxyItemize}
